# Si buscas colaborar 🤝
- Hola papu gracias por querer mejorar este glosario! Si quieres hacerlo estos son los pasos.
1. Haz un **Fork** del proyecto.
2. Crea una **Rama** para tu aporte (`git checkout -b feature/nuevo-atajo`).
3. Haz un **Commit** con tus cambios.
4. Abre un **Pull Request**.

Cualquier comando interesante ya sea de algun lenguaje, de Windows, Winux, o cualquier cosa por el estilo es bienvenida! Pero mantegamos todo en md para que tenga estilo.
