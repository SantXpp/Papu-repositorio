# Comandos Windows y terminal 🪟

## Atajos de Teclado ⌨️ 
* 'Win + V' Historial de Portapapeles. Para ver todo lo que has copiado antes (no solo lo último).
* 'Win + Shift + S' : Herramienta de recorte. Captura una parte de la pantalla y la guarda como imagen.
* 'Win + E' Abrir el explorador de archivos al instante.
* 'Win + D' Minimizar todo y mostrar el escritorio (presiona de nuevo para volver).
* 'Win + L' Bloquear la sesión.
* 'Ctrl + Shift + Esc' Abrir el Administrador de Tareas directamente.
* 'Win + Tab' Ver todos los escritorios virtuales y ventanas abiertas.
* 'Ctrl + Shift + N' Crear una carpeta vacia.

## Comandos de Terminal (CMD / PowerShell) 🖥️
*Usa estos comandos cuando no estés en Ubuntu y necesites moverte en Windows:*

| Acción | Comando Windows | Equivalente Linux |
| Ver archivos | 'dir' | 'ls' |
| Limpiar pantalla | 'cls' | 'clear' |
| Moverse de carpeta | 'cd [nombre]' | 'cd [nombre]' |
| Ver tu dirección IP | 'ipconfig' | 'ip a' |
| Probar conexión | 'ping [url]' | 'ping [url]' |
| Crear carpeta | 'mkdir [nombre]' | 'mkdir [nombre]' |
| Ver procesos activos | 'tasklist' | 'top' |
| Cerrar proceso a la fuerza | 'taskkill /F /IM [nombre.exe]' | 'kill -9' |

## Comandos de Reparación/Red (Nivel Admin) 🛠️
* 'systeminfo' : Muestra toda la info técnica de la PC (RAM, procesador, etc).
* 'getmac' : Muestra la dirección física (MAC) de tu tarjeta de red.
* 'netstat -an' : Muestra qué puertos están abiertos (útil para desarrollo web).
