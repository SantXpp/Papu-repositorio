#Comandos de Git

- 'git init'	"Despierta" a Git en tu carpeta. Solo se hace una vez al inicio.

- 'git clone' [url]	Descarga un proyecto de internet a tu PC.}

- 'git status'	Te dice qué archivos cambiaste y cuáles te falta guardar.

- 'git add .'	Prepara todos los cambios para la foto (el "commit").

- 'git add [archivo]' Prepara un archivo especifico para la foto

- 'git commit -m "mensaje"'	Toma la foto oficial. El mensaje debe ser claro (ej. "Agregué menú").

- 'git push origin main'	Sube tus fotos (commits) a la nube de GitHub.

- 'git pull'	Trae los cambios que alguien más subió a la nube.

- 'git log --oneline'	Muestra tu historial de fotos de forma resumida y elegante.
